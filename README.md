# 个人博客系统后端


## 项目简介


这是一个使用Go语言、Gin框架和GORM库开发的个人博客系统后端，实现了用户认证、文章管理和评论功能。


## 技术栈


- Go 1.21+

- Gin Web Framework

- GORM (ORM库)

- MySQL (数据库)

- JWT (用户认证)

- Logrus (日志记录)


## 运行环境


- Go 1.21+

- MySQL 5.7+
